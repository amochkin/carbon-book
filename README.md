# Carbon-based documentation theme for Hugo

Based on [IBM Carbon Design System](https://carbondesignsystem.com/) 

Inspired by [Hugo Book Theme](https://github.com/alex-shpak/hugo-book)

## Features:

- [ ] RTL support
- [ ] LaTeX/KaTeX support
- [ ] Buttons
- [ ] Badges
- [ ] Columns
- [ ] Accordions
- [ ] Hints/Quotes
- [ ] Tabs component
